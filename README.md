# Sistema de cadastro de produtos 

> status do projeto: Em desenvolvimento 

Para rodar esse projeto no seu computador, por favor digite no terminal:

```
npm install react 
``` 

Agora eu vou escrever um código rm HTML para testar 

```
<!doctype html>
<html>
   <head>
      <meta charset="utf-8"/>
      <title>Sistema</title>
    </head>
    <body>
      <p> Hello World! <p>
    </body>
  </html>
  ```
